ASSIGNMENT2

In this assignment, I encountered a significant challenge where the input box and the "Text" label consistently failed to align on the same line. Ultimately, I resolved this issue by implementing a selective vertical layout approach. Through this assignment, I learned how to edit .html files using CSS. I primarily acquired various techniques by browsing related CSS language posts published by other bloggers on the CSDN website.

ASSIGNMENT3

In this assignment, I used the CSS class called "hidden" to make the modal box show and hide, and when users create posts, I check that all required fields are filled; to avoid security issues, I use safe DOM methods like document.createElement() to make new HTML elements, element.textContent to add text content, element.setAttribute() to set attribute values, and element.appendChild() to build the element structure. When getting filter conditions, I also set text from the input to lowercase for case-insensitive matching, and for the minimum and maximum price inputs, empty values get default handling. When users create a post with a new city, the system gets all existing city options from the dropdown menu, compares the new city with them without caring about uppercase or lowercase letters, and if the city is not already in the menu, it adds it.
